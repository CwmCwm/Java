### 查看顺序
1. java-grammar     java语法，基础的东西，就是日常写业务代码后，就忘了一些语法（因为日常开发不用，大脑不会强化记忆）       
2. java-thread      java线程，基础，如何有效利用计算机硬件资源，解决瓶颈     
3. java-jvm         jvm，通过java示例代码去理解jvm的流程/思想/设计，其实你写java代码只需理解jvm    
4. java-algorithm   算法，这里只是用java去实现算法
